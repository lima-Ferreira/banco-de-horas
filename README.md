---

## Tecnologias utilizadas

- **Frontend**: React, Vite, Tailwind CSS, React Router
- **Backend**: Node.js, Express, MongoDB, Mongoose, JWT, bcryptjs
- **Outras**: Axios (ou fetch), html2pdf.js, jspdf

---

## Pré-requisitos

- Node.js >= 18
- npm ou yarn
- MongoDB (local ou Atlas)
- Git
